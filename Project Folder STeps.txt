Install Node.js
	https://nodejs.org/en/download

Creat enew project folder in vs code, then in bash terminal enter "npm init -y"
	creates package.json file

Install React/TS
	npm install react react-dom
	npm install typescript @types/react @types/react-dom --save-dev

Babel/Webpack Config (Bundles + Compiles code to local server)
	npm install --save-dev @babel/core @babel/preset-env @babel/preset-react @babel/preset-typescript babel-loader
	npm install --save-dev webpack webpack-cli webpack-dev-server style-loader css-loader
	npm install --save-dev html-webpack-plugin

Add code to '.babelrc' file (babel config)
Add code to 'webpack.config.js' file (webpack config)

Create src folder for project files
	index.html
	index.tsx (React app entry point)
	Login.tsx (Login entry on site)
	api.ts (For api calls)

To start local server in bash terminal on vscode enter "npm start". To close hit ctrl+C
